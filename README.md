# Lanka Market Pulse

Sri Lanka's premier financial news platform built with Next.js 14, deployed to AWS S3 + CloudFront.

## Features

- **Real-time Market Data**: Colombo Stock Exchange updates, ASPI, S&P SL20
- **AI-Powered Briefings**: Daily market summaries generated by AI
- **Comprehensive News**: Stock market, business, and economic news
- **PDF Reports**: On-demand report generation
- **Dark/Light Mode**: Full theme support
- **SEO Optimized**: Sitemap, RSS, OpenGraph, structured data
- **AdSense Ready**: Google AdSense integration
- **Analytics**: Google Analytics integration
- **Mobile Responsive**: Optimized for all devices
- **Fast Loading**: Static export with CloudFront CDN

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Lucide React icons
- **Charts**: Recharts
- **Search**: Fuse.js
- **PDF**: jsPDF
- **Deployment**: AWS S3 + CloudFront
- **Automation**: n8n workflows

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- AWS CLI (for deployment)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/lanka-market-pulse.git
cd lanka-market-pulse

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
# Generate static export
npm run build

# Generate sitemap and RSS
npm run generate-sitemap
```

### Deploy to AWS

```bash
# Configure AWS CLI first
aws configure

# Update deploy.sh with your bucket and distribution ID
# Then run:
./deploy.sh
```

## Content Management

Content is managed through Markdown files in the `content/` directory. Articles use frontmatter for metadata:

```markdown
---
title: "Article Title"
excerpt: "Brief description"
category: "stock-market"
author: "Author Name"
publishedAt: "2026-06-06T08:30:00Z"
tags: ["tag1", "tag2"]
featured: true
breaking: false
---

Article content in Markdown...
```

## Automation with n8n

The platform is designed to work with n8n automation workflows:

1. **RSS Feed Collection**: Collect news from multiple sources
2. **AI Summarization**: Use LLM APIs to summarize articles
3. **SEO Generation**: Auto-generate headlines and meta descriptions
4. **Content Publishing**: Generate Markdown files and push to repository
5. **Report Generation**: Create daily/weekly/monthly reports
6. **S3 Deployment**: Trigger build and deploy pipeline

## Environment Variables

Create a `.env.local` file for local development:

```env
# Google Analytics
NEXT_PUBLIC_GA_ID=G-YOUR_GA_ID

# Google AdSense
NEXT_PUBLIC_ADSENSE_ID=ca-pub-YOUR_ADSENSE_ID

# Site URL
NEXT_PUBLIC_SITE_URL=https://lankamarketpulse.com
```

## SEO Configuration

Update these values in `src/lib/seo.ts`:

- `SITE_NAME`: Your site name
- `SITE_URL`: Your domain
- `SITE_DESCRIPTION`: Default meta description
- `TWITTER_HANDLE`: Your Twitter handle
- Google verification code in `metadata.verification.google`

## Performance

The site is optimized for performance:

- Static HTML export (no server required)
- CloudFront CDN for global distribution
- Lazy loading images
- Optimized fonts with `next/font`
- Minimal JavaScript bundle
- CSS purging with Tailwind
